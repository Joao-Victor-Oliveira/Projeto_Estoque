# Projeto_Estoque

## Criação do projeto:

`python3 -m venv venv`

`source venv/bin/activate`

`pip install django`

`django-admin startproject projeto .`

`python manage.py startapp estoque`

`python manage.py runserver`


## Execução do projeto:

`python3 -m venv venv`

`source venv/bin/activate  (windows: .\venv\Scripts\activate)`

`pip install django`
